# VoiceToText Bot

**Telegram-бот для распознавания речи из аудио/видео файлов с преобразованием в форматированный текст.**

Бот принимает голосовые сообщения, видеозаписи и медиафайлы через Telegram, распознаёт речь, разделяет по спикерам (диаризация) и возвращает красиво отформатированный текст с кратким содержанием.

## Возможности

- 🎤 Распознавание речи из аудио и видео файлов
- 👥 Автоматическое разделение по спикерам (диаризация)
- 📝 Красивое форматирование с временными метками
- 📊 Автоматическое краткое содержание (summary)
- 🌐 Автоопределение языка (русский, английский и др.)
- 🔐 Авторизация пользователей через админа
- 📄 Экспорт длинных расшифровок в текстовый файл

## Технологии

- **Python 3.11+** / aiogram 3 — Telegram Bot Framework
- **AssemblyAI** — Speech-to-Text + Speaker Diarization
- **OpenRouter** — LLM для форматирования и summary
- **FFmpeg** — извлечение аудио из видео
- **SQLite** — хранение пользователей и истории
- **Docker** — контейнеризация

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/combx/Voice-To-Text.git
cd Voice-To-Text

# Скопировать и заполнить конфигурацию
cp .env.example .env
# отредактировать .env — указать токены и ключи

# Запустить через Docker
docker-compose up -d
```

## Конфигурация

Скопируйте `.env.example` в `.env` и заполните:

```env
BOT_TOKEN=           # Токен бота от @BotFather
ADMIN_ID=            # Ваш Telegram User ID
ASSEMBLYAI_API_KEY=  # API-ключ от assemblyai.com
OPENROUTER_API_KEY=  # API-ключ от openrouter.ai
```
