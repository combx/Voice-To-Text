Умный и мощный Telegram-бот, который превращает ваши аудио и видео сообщения в идеально структурированный текст. 

Забудьте о сплошных "простынях" нечитаемого текста без знаков препинания. VoiceToText Bot делает настоящую магию: он **узнаёт голоса разных людей**, **расставляет абзацы и запятые**, и даже пишет **краткое резюме** (summary) всего разговора!

---

## ✨ Как это выглядит?

Отправьте боту голосовое сообщение, видео-кружочек, запись звонка или подкаст (до 2 часов!). Бот вернёт вам красоту:

```text
📝 Расшифровка аудио
━━━━━━━━━━━━━━━━━━
📁 Файл: Голосовое сообщение
⏱ Длительность: 12:34
🗣 Спикеров: 2
🌐 Язык: Русский
🤖 Формат: gemma-3-27b-it
━━━━━━━━━━━━━━━━━━

🔵 Спикер 1 [00:00:00]
Привет! Как продвигается разработка нашего нового бота? Есть какие-то трудности?

🟢 Спикер 2 [00:00:06]
Привет. Выкатил первую версию, вроде всё отлично. Единственная проблема — длинные аудио. Если файл больше 20 мегабайт, Telegram API ругается. 

🔵 Спикер 1 [00:00:15]
Понял. Давай настроим локальный Telegram Bot API Server, это должно снять лимиты.

📌 Краткое содержание:
Обсуждение статуса разработки бота. Первая версия готова, но есть проблема с ограничением размера файлов (20 МБ) в Telegram API. Предложено решение: использовать локальный Telegram Bot API Server для обхода лимитов.
```

## 🚀 Суперспособности (Ключевые фичи)

* 👥 **Диаризация спикеров:** Бот понимает, кто и когда говорит, разделяя речь по цветам и номерам спикеров. Идеально для интервью и совещаний.
* ✍️ **Умное LLM-форматирование:** Нейросети из OpenRouter расставляют правильную пунктуацию, бьют текст на логические абзацы и делают Summary.
* 🛡 **Строгий Verbatim:** Нейросеть настроена так, чтобы **не удалять и не менять ни единого слова** из оригинальной расшифровки. Только структура и знаки препинания!
* 🌐 **Мультиязычность:** Бот автоматически определяет язык на записи (русский, английский, испанский и др.) и корректно его обрабатывает.
* ⏱ **Heartbeat-система:** Во время долгого распознавания бот каждые 30 секунд сообщает вам, что он жив и работает. Вы не будете гадать "завис он или нет?".
* 💰 **Трекинг баланса:** Бот сам считает, сколько кредитов в AssemblyAI вы потратили, и показывает примерный остаток баланса в часах и долларах.
* 📄 **Экспорт длинных текстов:** Если итоговый текст не влезает в лимиты Telegram, бот аккуратно пришлёт его отдельным `.txt` документом.
* 🔐 **Закрытый клуб:** Бот защищён административной панелью. Новые пользователи должны отправить "Запрос на доступ", который вы одобряете нажатием одной кнопки.

## 🛠 Под капотом (Стек технологий)

* Фреймворк: **aiogram 3.x**
* Извлечение аудио: **FFmpeg** (на лету вытаскивает звук прямо из видео `.mp4`, `.mkv`, кружочков)
* Транскрипция: **AssemblyAI REST API** (Universal-2 модель, лучшая точность и диаризация)
* Форматирование: **OpenRouter API** (Динамический fallback-chain из бесплатных моделей: *Gemma-3-27b, Qwen-3-32b* и др.)
* База данных: **SQLite (aiosqlite)** для хранения пользователей, статистики и чеков.
* Инфраструктура: **Docker & Docker Compose** для деплоя в одну команду. Сетевой режим `host` решает все проблемы с MTU на VPN-серверах.

## ⚙️ Установка и Запуск

Вам понадобится сервер с установленным `docker` и `docker-compose`.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/combx/Voice-To-Text.git
   cd Voice-To-Text
   ```

2. **Настройте окружение:**
   ```bash
   cp .env.example .env
   # Откройте .env в любом редакторе (nano/vim) и заполните токены
   nano .env
   ```

3. **Запустите магию:**
   ```bash
   docker-compose up -d --build
   ```

### 🔑 Где взять API ключи?

* `BOT_TOKEN` — у [@BotFather](https://t.me/BotFather) в Telegram. Узнайте свой `ADMIN_ID` у бота вроде @userinfobot.
* `ASSEMBLYAI_API_KEY` — зарегистрируйтесь на [AssemblyAI](https://www.assemblyai.com/). Дают $50 бесплатных кредитов при старте (хватит на ~185 часов аудио!). Привязка карты не нужна.
* `OPENROUTER_API_KEY` — зарегистрируйтесь на [OpenRouter.ai](https://openrouter.ai/). Бесплатные модели работают "из коробки" без пополнения баланса.

## 👨‍💻 Администрирование

Будучи добавленным как `ADMIN_ID`, вы обладаете полным контролем:
* `/users` — посмотреть список всех, кто пользуется (или стучится в) бота.
* `/approve <USER_ID>` — выдать доступ.
* `/revoke <USER_ID>` — забрать доступ.
* `/stats` — посмотреть статистику: сколько часов аудио было расшифровано сервером за всё время.
